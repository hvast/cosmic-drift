# 🚀 命令行启动指南

## 📋 快速命令参考

### 方式一：自动清理端口后启动（推荐）

```bash
# 启动完整环境（前端 + 后端）- 自动清理 3000 和 3001 端口
npm start

# 仅启动后端 - 自动清理 3001 端口
npm run start:backend

# 仅启动前端 - 自动清理 3000 端口
npm run start:frontend
```

### 方式二：手动清理端口

```bash
# 清理所有端口（3000 和 3001）
npm run kill:ports

# 然后启动服务
npm run dev              # 启动前端+后端
npm run dev:backend      # 仅启动后端
npm run dev:frontend     # 仅启动前端
```

### 方式三：直接运行脚本

```bash
# 使用 Node.js 脚本（不推荐直接调用，请使用 npm 命令）
node scripts/clean-start.js          # 启动完整环境
node scripts/clean-start.js backend  # 仅启动后端
node scripts/clean-start.js frontend # 仅启动前端
node scripts/kill-ports.js           # 清理端口

# 使用批处理脚本（Windows）
.\scripts\start-dev.bat       # 启动完整环境（需手动清理端口）
.\scripts\install-all.bat     # 安装所有依赖
```

## 🎯 命令对比

| 命令 | 自动清理端口 | 启动内容 | 适用场景 |
|-----|------------|---------|---------|
| `npm start` | ✅ 3000+3001 | 前端+后端 | **最常用** |
| `npm run start:backend` | ✅ 3001 | 仅后端 | 后端开发 |
| `npm run start:frontend` | ✅ 3000 | 仅前端 | 前端开发 |
| `npm run dev` | ❌ | 前端+后端 | 端口已清理 |
| `npm run kill:ports` | ✅ 清理所有 | 无 | 清理端口 |

## 💡 使用示例

### 场景 1：日常开发（最简单）

```bash
# 一条命令搞定一切
npm start
```

这会自动：
1. 检查并清理 3000 和 3001 端口
2. 启动后端服务器 (http://localhost:3001)
3. 启动前端服务器 (http://localhost:3000)

### 场景 2：只开发后端

```bash
npm run start:backend
```

### 场景 3：只开发前端

```bash
npm run start:frontend
```

### 场景 4：端口被占用无法启动

```bash
# 先清理端口
npm run kill:ports

# 然后启动
npm start
```

### 场景 5：使用自定义端口清理工具

```bash
# 清理特定端口
.\scripts\kill-port.bat 3001    # 清理 3001 端口
.\scripts\kill-port.bat 3000    # 清理 3000 端口
```

## 📝 创建自定义别名（可选）

### PowerShell Profile

编辑 `$PROFILE` 文件：

```powershell
# 打开配置文件
notepad $PROFILE

# 添加以下别名
function start-cosmic { npm start }
function start-be { npm run start:backend }
function start-fe { npm run start:frontend }
function kill-ports { npm run kill:ports }
```

保存后，可以使用：

```powershell
start-cosmic  # 启动完整环境
start-be      # 启动后端
start-fe      # 启动前端
kill-ports    # 清理端口
```

### Git Bash

编辑 `~/.bashrc`：

```bash
# 添加别名
alias cosmic='npm start'
alias cosmic-be='npm run start:backend'
alias cosmic-fe='npm run start:frontend'
alias kill-ports='npm run kill:ports'
```

然后运行 `source ~/.bashrc` 使其生效。

## 🚨 常见问题

### Q: 端口清理失败怎么办？

```bash
# 使用管理员权限运行 PowerShell 或 CMD
# 然后执行：
npm run kill:ports
```

### Q: 想查看哪个进程占用了端口？

```bash
# Windows
netstat -ano | findstr :3001
netstat -ano | findstr :3000
```

### Q: 如何后台运行服务？

```bash
# Windows PowerShell
Start-Process npm -ArgumentList "start" -WindowStyle Hidden

# 或使用 pm2（需要先安装）
npm install -g pm2
pm2 start npm --name "cosmic-drift" -- start
```
