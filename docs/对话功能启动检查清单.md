# 对话功能启动检查清单

## 准备工作

### 1. 环境配置
- [ ] 确认 Node.js 已安装（18+）
- [ ] 确认 MySQL 已运行
- [ ] 确认已安装项目依赖（`npm install`）

### 2. API配置
- [ ] 获取 OpenAI API Key
- [ ] 在 `backend/.env` 中配置 `OPENAI_API_KEY`
- [ ] 如果没有API Key，系统会使用降级方案（预设回复）

### 3. 数据库迁移
- [ ] 运行 `cd backend && npm run migrate`
- [ ] 确认创建了 `conversations` 表
- [ ] 确认创建了 `messages` 表

## 启动服务

### 后端
```bash
cd backend
npm run dev
```
- [ ] 后端启动成功（端口3001）
- [ ] 看到 "Server running on port 3001"
- [ ] 没有数据库连接错误

### 前端
```bash
cd frontend
npm start
```
- [ ] 前端启动成功（端口3000）
- [ ] 浏览器自动打开
- [ ] 没有编译错误

## 常见问题排查

### 后端无法启动
- [ ] 检查端口3001是否被占用
- [ ] 检查数据库连接配置
- [ ] 查看控制台错误信息

### 前端无法启动
- [ ] 检查端口3000是否被占用
- [ ] 运行 `npm install` 重新安装依赖
- [ ] 清除缓存 `npm cache clean --force`

### AI不回复
- [ ] 检查 OPENAI_API_KEY 是否配置
- [ ] 检查网络连接
- [ ] 查看后端控制台错误
- [ ] 如果API不可用，应该看到降级回复

### 对话历史不显示
- [ ] 确认数据库迁移成功
- [ ] 检查浏览器控制台错误
- [ ] 确认已登录
- [ ] 刷新页面重试

### 情绪值不变化
- [ ] 检查后端日志
- [ ] 使用明确的积极/消极词汇
- [ ] 刷新页面查看
